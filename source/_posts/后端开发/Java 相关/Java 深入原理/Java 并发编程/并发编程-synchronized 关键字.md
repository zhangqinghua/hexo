`synchronized` 是 Java 中解决并发问题的一种最常用的方法，也是最简单的一种方法。

`synchronized` 的作用主要有三个：
1. 确保线程互斥的访问同步代码
1. 保证共享变量的修改能够及时可见
1. 有效解决重排序问题。
